# film-inspired-games

영화 모티브 기반 2D 인터랙션 게임 실험 프로젝트.

첫 방향은 영화 `버닝`, `양들의 침묵`에서 가져온 심리 긴장감과 `Florence`식 짧은 터치 상호작용을 결합한 프로토타입.

## 개발 환경

- Unity: `6000.0.67f1`
- 템플릿: Universal 2D
- 목표 화면: 세로 9:16
- 기본 시작 씬: `Assets/Scenes/SampleScene.unity`

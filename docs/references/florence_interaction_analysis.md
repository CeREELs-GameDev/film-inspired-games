## 세로 화면용 연출 기준

- 한 장면의 중심은 얼굴, 손, 물건 중 하나만 선택
- 손이 하는 일은 화면 아래 3분의 1 안에서 끝나게 배치
- 인물 사이 거리는 위아래 높이 차이와 빈 공간으로 표현
- 말풍선은 화면 중앙을 오래 막지 않게 하고, 인물의 얼굴 옆으로 빠르게 이동
- 넓은 장소는 세로로 길게 보여주기보다 한 물건 또는 한 구역을 잘라 보여주기
- 색 변화는 배경 전체보다 기억, 위험 물건, 특정 인물에 먼저 적용

## 첫 프로토타입 제안

### 장면: 낯선 남자의 물건 상자

1. 책상 위 상자를 열어 사진, 영수증, 테이프 3개를 꺼냄
2. 물건을 테이블 위 세 구역에 끌어놓으면 인물의 짧은 기억 장면이 보임
3. 어느 위치에 두어도 이야기는 진행. 다만 장면의 색, 대사 간격, 마지막 표정이 달라짐
4. 마지막에는 상자를 닫거나 하나의 물건을 손에 남기는 선택만 제공

이 장면은 배경 에셋이 적어도 구현 가능. 상자, 물건 3개, 손, 얼굴 2개, 기억 배경 3장으로 시작 가능.
